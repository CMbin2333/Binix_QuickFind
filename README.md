# Binix_QuickFind (速搜) 🔍

一款轻量级、响应迅速的 Chrome 浏览器扩展，支持在一个美观的弹窗中一键综合搜索和管理您的书签、历史记录与下载内容。

## ✨ 核心功能

* **全局与分类搜索**: 无缝切换“书签”、“历史记录”、“下载”标签页，或在“综合”面板中一次性搜索所有内容。
* **全键盘极速操作**: 彻底告别鼠标。支持方向键上下导航、左右折叠展开，并可通过快捷键直接执行打开或删除操作。
* **智能日期分组**: 搜索结果按日期（如今天、昨天、星期几）自动分组归类并支持折叠，界面清晰不杂乱。
* **书签热度追踪**: 自动记录书签的访问次数，并支持按访问频次高低排序（可在设置中随时切换或清空）。
* **快捷悬浮操作**: 选中或悬浮在结果项上即可快速执行：
    * 📋 复制链接
    * 📁 打开所在文件夹（针对下载文件）
    * 🔄 重新下载（针对取消或中断的任务）
    * 🗑️ 删除记录（内置防误删二次确认弹窗）
* **暗色模式与个性化**: 原生支持亮色/暗色主题一键切换，并支持在设置面板中自定义插件的主题强调色。

## ⌨️ 快捷键指南

| 快捷键 | 功能说明 |
| :--- | :--- |
| `Ctrl + B` | 唤醒 BinixOvO 搜索弹窗 |
| `↑` / `↓` | 在搜索结果中上下导航 |
| `←` / `→` | 折叠 / 展开当前日期分组 |
| `Enter` | 打开选中项，或展开/折叠当前标题组 |
| `Del` | 删除当前选中项 |
| `Tab` | 在不同的搜索分类标签之间切换 |
| `Esc` | 关闭确认弹窗、设置面板或退出扩展 |

## 🚀 安装指引

1. 下载或克隆本仓库到本地。
2. 打开 Google Chrome 浏览器，地址栏输入 `chrome://extensions/` 进入扩展程序页面。
3. 开启页面右上角的 **开发者模式**。
4. 点击 **加载已解压的扩展程序**，选择包含 `manifest.json` 等文件的项目文件夹。
5. 将 **BinixOvO** 固定到浏览器工具栏，随时使用 `Ctrl+B` 召唤它！

## 🔒 权限与隐私声明

本扩展完全在本地运行，**不会**收集您的任何隐私数据或将其发送至第三方服务器。所需权限说明如下：

* `bookmarks`: 用于搜索、读取和删除您的书签记录。
* `history`: 用于搜索和清理您的浏览历史。
* `downloads`: 用于检索下载列表、打开文件目录或重新启动下载。
* `tabs`: 用于在新标签页中打开您的搜索结果。
* `storage`: 用于在本地保存您的偏好设置、主题颜色及书签访问统计数据。
